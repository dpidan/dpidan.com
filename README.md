## About This Repo

The old site was on an ancient technology stack and had become a bit of a chore to maintain. I needed to replace it but didn't have the time. I replaced it with a quick hand-coded static HTML page with very basic contact information.

It is slowly evolving - with some help from handy AI tools - to include a profile page and a little about how I work. The site is built with vanilla HTML, CSS and JavaScript - no frameworks or libraries and no build steps. It's not intended to be a technical showcase by any means but merely a way to say a little more than I can on LinkedIn or GitHub.
